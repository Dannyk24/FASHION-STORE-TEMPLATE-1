# Fashion Store Landing Page

A modern, responsive fashion store landing page built with HTML, CSS, and JavaScript. The project is designed for fashion brands, boutiques, and clothing vendors looking to showcase products and drive customer inquiries through WhatsApp.

## 🌐 Live Demo

🔗 Live Site: https://fashion-store-template-zeta.vercel.app/

## ✨ Features

- Fully responsive design
- Modern and clean user interface
- Product showcase section
- Featured collections
- Mobile-friendly navigation
- WhatsApp integration for customer inquiries
- Smooth scrolling experience
- Fast loading performance
- Built with pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📱 Responsive Design

The website is optimized for:

- Mobile devices
- Tablets
- Laptops
- Desktop screens
